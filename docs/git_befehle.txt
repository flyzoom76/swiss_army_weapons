# Sicherstellen dass du auf develop bist
git checkout develop

# Änderungen speichern und hochladen
git add .
git commit -m "Beschreibung"
git push

# Git -> Lokal
git pull origin develop


Oder nur bestimmte Dateien hochladen:
# Nur eine bestimmte Datei
git add dateiname.cpp

# Nur einen Ordner
git add data/

# Dann commit + push wie oben

Schnell-Übersicht was git macht:
Befehl	Was es tut
git status	Zeigt welche Dateien neu/geändert sind
git add .	Markiert alle Dateien zum Hochladen
git commit -m "..."	Speichert einen Snapshot mit Beschreibung
git push	Lädt alles zu GitHub hoch
